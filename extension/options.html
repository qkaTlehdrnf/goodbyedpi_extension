GoodbyeDPI for Chrome — Settings

This proxy applies to Chrome only. Other apps are not affected.
Address and port of the local ByeDPI (ciadpi) SOCKS5 proxy. Default 127.0.0.1:1080.
Pick a preset or type your own. The combination that works varies by ISP and region.
All sites: send all Chrome traffic through the proxy. List: only the domains below (everything else connects directly).
Checked (recommended): install the companion app once, then the toggle launches and stops ciadpi for you. Unchecked (manual): the extension only sets the proxy — you start ciadpi yourself (Windows ciadpi.exe -i 127.0.0.1 -p 1080 …, macOS/Linux ./ciadpi -i 127.0.0.1 -p 1080 …). If the proxy isn't running when you toggle on, the popup shows install options instead of breaking your connection.
