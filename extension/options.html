GoodbyeDPI for Chrome — 설정

이 프록시는 Chrome 트래픽에만 적용됩니다. 다른 앱은 영향을 받지 않습니다.
로컬 ByeDPI(ciadpi) SOCKS5 프록시의 주소·포트입니다. 기본값 127.0.0.1:1080.
아래 프리셋을 누르거나 직접 입력하세요. ISP·지역마다 잘 되는 조합이 다릅니다.
전체: 모든 Chrome 트래픽을 프록시로. 목록: 아래 도메인만 프록시로(나머지는 직접 연결).
체크 해제 시 프록시 설정만 바꾸며, ciadpi 는 직접 실행해야 합니다 (backend\start-ciadpi.bat).
